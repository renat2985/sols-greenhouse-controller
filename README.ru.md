# 🌱 Sols Greenhouse Controller

Язык: [English](README.md) | Русский

Умный контроллер теплицы на базе сенсорного ESP32-S3 дисплея `JC3248W535`.

Небольшой экран превращается в панель управления теплицей: климат, полив, вентиляция, облачные графики и данные, которые удобно анализировать с AI-помощником.

В этом репозитории лежат публичная документация, схема подключения, Gerber-файлы платы и бинарники прошивки. Исходный Arduino/ESP32 код не публикуется.

## ✨ Что умеет устройство

- 📺 Показывает данные теплицы на 3.5-дюймовом сенсорном экране.
- 📡 Подключается к Wi-Fi из файла `config.txt` на MicroSD.
- ☁️ Отправляет телеметрию на [sols.lv](https://sols.lv).
- 📱 Показывает QR-код для облачной панели и локальной настройки.
- ⚡ Управляет тремя реле:
  - вытяжной вентилятор,
  - приточный вентилятор,
  - кран воды или насос.
- 🌡️ Сравнивает температуру и влажность внутри/снаружи теплицы.
- 💧 Поддерживает полив по расписанию или по датчику влажности почвы.
- 📈 Записывает данные датчиков и реле на MicroSD.

## 🚀 Рекомендуемая установка

Самый простой способ прошить устройство - веб-установщик:

👉 [https://sols.lv/upgrade](https://sols.lv/upgrade)

Он автоматически загружает нужные firmware-файлы. Большинству пользователей не нужно выбирать `.bin` файлы вручную.

## 🧩 Оборудование

Основной экран:

- 🖥️ JC3248W535 ESP32-S3 touch display
- дисплей 320x480
- MicroSD карта, FAT16 или FAT32
- ESP32-S3 с PSRAM

Датчики:

- 🌡️ внутри: модуль AHT20 + BMP280 по I2C
- 🌤️ снаружи: модуль SHT40 по I2C
- 🌱 опционально: аналоговый датчик влажности почвы
- ☀️ опционально: LDR-датчик света

Выходы:

- ⚡ 3 канала реле для вытяжки, притока и воды

Схема подключения описана в [HARDWARE.ru.md](HARDWARE.ru.md).

## 🛠️ Плата реле и датчиков

В репозитории есть Gerber-архив:

📦 [Gerber.zip](Gerber.zip)

Его можно загрузить в сервис производства плат, например JLCPCB, PCBWay или другой сервис, который принимает Gerber. Плата задумана как дополнительная плата реле/датчиков для контроллера теплицы.

Перед заказом прочитайте [PCB.ru.md](PCB.ru.md).

## 🔥 Файлы прошивки

Бинарники лежат в папке [firmware](firmware/).

Предпочтительный вариант - multi-file bundle:

| Файл | Адрес записи |
| --- | ---: |
| `greenhouse.bootloader.bin` | `0x0` |
| `greenhouse.partitions.bin` | `0x8000` |
| `boot_app0.bin` | `0xe000` |
| `greenhouse.app.bin` | `0x10000` |

Также есть `greenhouse.full.bin` для инструментов, которые умеют записывать один общий образ.

Параметры сборки:

- Board: ESP32S3 Dev Module
- Flash size: 16 MB
- Partition scheme: No OTA, 2 MB APP / 2 MB SPIFFS
- PSRAM: OPI PSRAM
- USB CDC on boot: enabled
- CPU: 240 MHz
- Flash mode: QIO 80 MHz в меню Arduino

## 🧾 Конфигурация на MicroSD

Создайте файл `/config.txt` в корне MicroSD карты.

Примеры:

- [config.example.txt](config.example.txt) - латиница/латышский стиль комментариев
- [config.example.ru.txt](config.example.ru.txt) - русская версия комментариев

## ☁️ Облачная панель

Страница устройства открывается по публичному токену:

```text
https://sols.lv/d/YOUR_PUBLIC_TOKEN
```

Для отправки телеметрии устройство использует приватный API-токен. Не публикуйте свой настоящий `cloud_api_token`.
